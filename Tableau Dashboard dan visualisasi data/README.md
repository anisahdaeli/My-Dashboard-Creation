# Tableau Dashboards

This folder contains a collection of data visualization projects created using Tableau.  
The dashboards focus on health analytics, sales performance analysis, and global data visualization.

---

## Folder Structure and Descriptions

### 1. Dashboard Diabetes
<img width="1366" height="768" alt="diabetes" src="https://github.com/user-attachments/assets/c8c49b1c-029a-4f44-9342-0d92469a7e5a" />

This folder contains dashboards related to diabetes analysis, including:
- Distribution of diabetes and non-diabetes cases by age
- Outcome distribution based on blood pressure
- Outcome analysis based on patient glucose levels

### 3. Covid-19 Distribution Dashboard
<img width="1366" height="768" alt="persebaran covid-19" src="https://github.com/user-attachments/assets/871df984-8725-47ca-a046-f067d4efe5c6" />
